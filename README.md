# YoutubeDownloader
Just a terminal way of downloading Youtube Vids on mp3.

Requirements:
modules:
  yt_dlp
  youtube-search-python
  ffmpeg

#Installation
```
use the classic git clone and clone this repo

for modules:

$ pip install yt_dlp
$ pip install youtube-search-python
$ pip install ffmpeg
```

#Run

```
cd YoutubeDownloader
python3 ytd.py
```
